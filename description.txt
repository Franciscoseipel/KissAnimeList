Features:
- Shows Mal Score (Link to Mal)
- Change Mal Data (Status/Episode/Rating)
- Auto track episodes
- Link to next episode
- Bookmark thumbnails
- Links from Mal to kiss and one click search
- Link to last streaming page on Myanimelists Animelist (Tags have to be activated)
- add/remove Kissanime.ru Bookmarks = plan to watch/dropped on Mal
- Keeps track of re watches
- Automatically sets beginning and finishing Date
- Uses browsers Mal session (Log in credentials never needed)
- Caches entries for faster loading times
- Should work with any Ajax episode loading user scripts ( <a href="https://greasyfork.org/de/scripts/27780-nokissreload">NoKissReload</a> )

v0.87:
- Materialize UI
- Add miniMAL popup


Usage: Just install the Script and be sure to be logged in on MyAnimeList, like you normally would.

Config: miniMAL

The first time executing the script, it will ask you to allow a domain. This is because there are many google domains (.com .en .de ...) and I didn't add all of them.
