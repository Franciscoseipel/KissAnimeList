<b>Autotracking:</b>
- autoincrement watched episodes/chapters
- Automatically adds beginning and finishing dates
- Manages Rewatches (MAL entry needs to be completed and episode/chapter one must be open)

<b>Myanimelist:</b>
<img src='https://raw.githubusercontent.com/Franciscoseipel/Mal-for-Kissanime-Greasymonkey-/master/Screenshots/Myanimelist_sidebar.png'>
- Adds oneklick search links
- Shows direct links using a Database.

(This uses MAL's Tag System)

<b>miniMAL:</b>
<img src='https://raw.githubusercontent.com/Franciscoseipel/Mal-for-Kissanime-Greasymonkey-/master/Screenshots/miniMAL.png'>
- Shows a small overview of the opened anime/manga using data from MAL
- Add the posibility to search for anime
- Shows your MyAnimeList Bookmarklist
- Holds the settings page

<b>ETC:</b>
- Uses browsers MAL session (Log in credentials never needed)
- Should work with any Ajax episode loading user scripts ( <a href="https://greasyfork.org/en/scripts/27780-nokissreload-captcha-skip">NoKissReload [CAPTCHA SKIP]</a> )

The first time executing the script, it will ask you to allow a domain. This is because there are many google domains (.com .en .de ...) and I didn't add all of them.

<b>Usage:</b>
Just install the Script and be sure to be logged in on MyAnimeList, like you normally would.

Tested with:
<a href="https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo?hl=en">Tampermonkey</a> (Chrome)

Android (Only works on <a href="https://play.google.com/store/apps/details?id=org.mozilla.firefox">Firefox</a> with the <a href="https://addons.mozilla.org/en-GB/android/addon/userunified-script-injector/">usi (User|Unified Script Injector)</a> extension)
